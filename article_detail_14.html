# 자바
## 연산자 - 연산자에는 대입, 산술, 비교, 논리 연산자 등이 있다 - 산술 연산자는 + - * / % 등이 있다
### 예시
```java int a = 10; a = a + 10; System.out.println(a); a = a - 10; System.out.println(a); a = a * 10; System.out.println(a); a = a / 10; System.out.println(a); a = a % 10; System.out.println(a); String b = "와" String c = "우" System.out.println(b + c); ``` ## 결과 20 10 100 10 1 와우 ## 부연 - 더하기 연산자는 문자열과 문자열을 합칠 때도 사용할 수 있다 - 정수 나누기 연산 시 실수 부분은 버려진다 (3 / 10 == 3.3333... ) -> 3 - 정수를 실수로 나눌 경우 자동 형 변환이 일어나고 자료형이 실수가 된다 -> 3.333...
