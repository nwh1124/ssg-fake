# 자바
## 반복문 - 특정 코드를 정해진 조건만큼 반복한다 - for, while 등이 있다
### 반복문 예시
```java // n까지의 합을 구하는 반복문 Scanner sc = new Scanner(System.in); // 정수 n을 입력받기 위한 스캐너 준비 int n = sc.nextInt(); // 정수 n을 입력 받는다 int sum = 0; // 합을 저장할 정수형 변수 선언 for(int i = 0; i < n; i++){ // 반복문 for을 통해 sum = sum + i; // n번만큼 덧셈을 반복한다 } // 반복문 종료 System.out.printf("0부터 %d까지의 합 = %d", n, sum); // 결과를 출력한다 ``` - 조건문 if 처럼 특정 조건이 참이라면 코드를 실행한다 - if와 다른 점으로는 조건이 참일 때 무한히 반복한다는 점이다 - 조건문을 적절히 이용할 경우 쓰임새가 매우 다양하다
