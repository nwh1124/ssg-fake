# 자바
## 자료형 변환 - 같은 값이라도 자료형 별로 bit는 다르게 저장된다
### 예시
```java int a = 200; float b = 200.0; ``` ## 결과 a bit == 00000000 00000000 00000000 11001000 b bit == 01000011 01001000 00000000 00000000 ## 부연 - 데이터의 손실이 없다면 자동 형 변환이 일어난다 - int a = 2; float b = a; .. b == 2; - 위의 경우는 정수형 -> 실수형으로 변환하는 과정에서 데이터 손실이 없기 때문에 자동적으로 자료형이 변한다 - 반대로 float을 int에 대입할 경우 에러가 난다 - 형 변환 규칙 - byte -> shot, char -> int -> long -> float -> double - 자동 형 변환이 일어나지 않는 경우도 데이터 앞에 (int)를 붙이는 것과 같은 방식으로 명시적 형 변환도 가능하다
