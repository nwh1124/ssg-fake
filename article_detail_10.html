# 자바
## 주석 데이터 타입#1 데이터 크기 - 데이터 크기의 최소 단위는 0과 1로 이루어진 bit이다 - byte는 bit*8의 크기를 가진다 - byte부터는 각 1024배씩 커지면서 크기 단위가 정해진다 - kilo, mega, giga, tera, peta, exa ... byte로 이어진다
### 예시
```java 8 bit = 1 byte 8,192 bit = 1024 byte = 1 kilobyte 8,388,608 bit = 1024kilobyte = 1megabyte ... ... ``` ## 결과 - ... ## 부연 - byte 단위부터는 1024 배 커질 때마다 용량의 단위가 바뀐다
