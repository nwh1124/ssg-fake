# 자바
## 변수#6 배경지식#2 문자열 - 자료형에는 문자를 담기 위한 String이 있다 - String은 덧셈 기호로 문자열을 합칠 수 있다
### 예시
```java String str = "맥주 한 캔 땡긴다"; a = 2.2; System.out,printf(str + a); ``` ## 결과 - 맥주 한 캔 땡긴다2.2 ## 부연 - 문자열형 변수에 숫자를 더할 경우 숫자가 그대로 문자열에 추가된다 - 숫자 - > 문자열로는 치환이 가능하다 - 반대는 성립하지 않는다
