# 자바
## 변수#4 배경지식#1 정수 - 변수에는 문자, 숫자 등 데이터를 담을 수 있다 - 일반적으로 명령어가 끝났을 때는 세미콜론(;)을 사용한다
### 예시
```java int a; a = 2; System.out,printf(a+1); ``` ## 결과 - 3
## 부연 - int a는 a를 정수형 변수로 선언한 것이다 - a = 2는 정수형 변쉐 a에 정수 2를 할당한 것이다 - a(=2)에 + 1한 값을 출력하므로 결과가 3이 나온다 - 정수형 변수는 주로 int가 쓰이지만 더 큰 값을 저장할 수 있는 long도 있다
