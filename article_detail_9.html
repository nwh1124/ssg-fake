# 자바
## 주석 - 주석은 // 코드, /* 코드 */ 식으로 사용된다 - 주석에 포함된 코드는 활성화되지 않는다 - 코딩 설명을 위해 쓰이며, 일시적으로 코드를 비활성화하기 위해서도 쓰인다
### 예시
```java String str = "난 치킨"; String not = "안"; String like = "좋아해"; // str = str + not; str = str + like; System.out,printf(str); ``` ## 결과 - 난 치킨 좋아해 ## 부연 - ctl + / 단축키로 커서가 있는 줄의 코드를 주석 처리할 수 있다 - ctl + shift + / 단축키로 드래그된 코드를 전부 주석처리할 수 있다
